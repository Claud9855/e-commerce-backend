# E-commerce Back End
----------------------

# Description
--------------
- This app is the backend service of a e-commerce website
- The backend uses express.js and sequelize to connect and interact with a mysql database

# Technologies
---------------
- node.js
- express.js
- mysql2
- sequelize
- Dotenv
